1. Install packages Using "npm install". It fetches all the dependencies into node_modules folder.
2. To Run this application for user in your localhost use "npm run start" command.
3. To Run this application for admin in your localhost use "npm run start_admin" command.
4. To create a bundle which will be deployed  use "npm run deploy" command
5. To create a admin bundle which will be deployed use "npm run deploy_admin" command.
